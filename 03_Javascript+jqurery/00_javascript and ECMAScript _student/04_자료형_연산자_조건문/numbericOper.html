Operators

1. Numberic operators (숫자연산자)

  • + : add
  • - : substract
  • / : divide
  • * : multiply
  • % : remainder
  • ** : exponentiation

2. String operators (문자결합)

3. Assignment operators (대입연산)

  • x = y
  • x += y  : x = x+y
  • x -= y  : x = x-y
  • x *= y  : x = x*y
  • x /= y  : x = x/y

4. Increment and Decrement operators (증감연산자)

  • ++ : 1씩 증가
  • -- : 1씩 감소

5. Comparison operators (비교연산)

  • <
  • <=
  • >
  • >=
  • == : 표기된 숫자만 일치하면 true
  • === : 자료형도 일치해야 true

6. Logical operators

  • || (or)
  • && (and)
  • ! (not)

7. Equallity

  • ===
  • !==

8. Ternary operators (삼항조건연산자)

  • ? : 간단하게 출력할 때 사용
  • let 변수 = 조건 ? 참 실행문 : 거짓 실행문
