반복문

while 문

do while 문

블록을 먼저 실행하고 조건문 읽음

for 문

  • for(시작값, 조건문, 증감식){실행문}
  • for(조건){document.getElementById('id값').innerHTML += 배열변수[i]}

forEach

  • foreach (반드시 배열 구조) 익스 11버전부터 사용 가능
  • 배열변수.forEach(function(item, index){배열변수[index]})
  • 배열변수.forEach(function(item, index){item})

for of

  • for(item of 배열변수){item}

for in

  • 배열에서 사용 안함. 객체 내의 값들마다 할일
  • for(x in 객체변수){x + 객체변수[x]}}

break 문

반복문을 완전히 끝내는 것

continue 문

반복문에서 지금 사용한 부분만 스킵하고 다른 것만 실행
